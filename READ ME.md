# Mosepele Tours Bus Booking System (Java)

## Description

A Java desktop application that allows users to log in and manage bus ticket bookings.
The system provides a graphical interface built using Java Swing where users can book tickets and view booked tickets.

## Features

* User login system
* Book bus tickets
* View booked tickets
* Graphical user interface

## Technologies

* Java
* Java Swing
* OOP concepts
* Event handling

## How to Run

1. Clone the repository
2. Compile the Java files
3. Run Main.java

Example:

javac *.java
java Main
